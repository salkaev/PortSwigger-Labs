# PortSwigger Web Security Labs

Automated exploit scripts for PortSwigger Web Security Academy labs. Each script demonstrates a specific web vulnerability and automates the exploitation process.

## Purpose

This repository serves as:
- **Learning resource** for web application vulnerabilities
- **Automation toolkit** for PortSwigger lab solutions
- **Reference** for common attack patterns (SQLi, SSRF, brute-force, etc.)

## Vulnerabilities Covered

| Vulnerability | Scripts |
|---------------|---------|
| **SQL Injection** | Blind SQLi with conditional errors, Blind SQLi with conditional responses |
| **SSRF** | Basic SSRF, SSRF with blacklist-based input filter |
| **Authentication** | Username enumeration (4 variants), Broken brute-force protection, 2FA broken logic |
| **Session Management** | Brute-forcing stay-logged-in cookie |
